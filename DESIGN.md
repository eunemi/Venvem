# Venvem: Design System

## Palette
- **Background**: `#000000` (Deep Space)
- **Surface**: `#050505` (Obsidian)
- **Border**: `rgba(255, 255, 255, 0.1)` (Ghostly White)
- **Primary**: `#0070F3` (Electric Blue)
- **Secondary**: `#7928CA` (Vivid Violet)
- **Accent**: `#00DFD8` (Cyber Teal)
- **Glow**: `rgba(0, 112, 243, 0.5)`

## Typography
- **Headings**: Satoshi / General Sans (Bold, Letter Spacing: -0.02em)
- **Body**: Inter / Satoshi (Normal, Line Height: 1.6)
- **Mono**: JetBrains Mono / SF Mono

## UI Patterns
- **Glassmorphism**: 
  - Background: `rgba(10, 10, 10, 0.7)`
  - Blur: `20px`
  - Border: `1px solid rgba(255, 255, 255, 0.1)`
- **Atmospheric Lighting**: 
  - Radial gradients behind key components.
  - Hover glow effects on cards and buttons.
- **Micro-interactions**: 
  - Smooth scale transitions on hover.
  - Magnetic button effects.
  - Text morphing for AI section transitions.

## Components to Build
- `InteractiveHero`: 3D elements + kinetic typography.
- `ProductGrid`: Luxury product cards with download stats.
- `BookingModule`: Minimalist scheduling interface.
- `AdminDashboard`: Streamlined content management.
